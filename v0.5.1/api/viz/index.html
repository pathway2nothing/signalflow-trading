    Visualization¶

    Interactive pipeline visualization for SignalFlow, inspired by Kedro-Viz.

    Module

    signalflow.viz -- Available as sf.viz.pipeline(), sf.viz.features(), sf.viz.serve().

    Quick Example¶

    import signalflow as sf

# Build a backtest pipeline
builder = (
    sf.Backtest("my_strategy")
    .data(raw=raw_data)
    .detector("example/sma_cross", fast_period=20, slow_period=50)
    .exit(tp=0.03, sl=0.015)
)

# Interactive HTML visualization
sf.viz.pipeline(builder)

# Mermaid diagram (for docs)
mermaid_code = sf.viz.pipeline(builder, format="mermaid", show=False)

# Local development server
sf.viz.serve(builder, port=4141)

    signalflow.viz.serve ¶

    serve(source, *, port: int = 4141, open_browser: bool = True)

    Start local visualization server (like Kedro-Viz).

    Starts an HTTP server at localhost:{port} serving the pipeline visualization. Press Ctrl+C to stop.

    Parameters:

    Name Type Description Default
    source

    Pipeline source (BacktestBuilder, FeaturePipeline, etc.)

    		 required
    port int

    Server port (default: 4141)

    		 4141
    open_browser bool

    Open browser automatically

    		 True
    Example

    import signalflow as sf builder = sf.Backtest("test").data(...).detector(...) sf.viz.serve(builder) # Opens http://localhost:4141
